# 格式化块设备并挂载使用

## Context

克隆软件源时遇到了空间不足的问题，为服务器添加块设备，格式化并挂载

---

## Key Takeaway

- 使用`fdisk`工具为块设备分区
- 使用`mkfs.ext4`将设备格式化为ext4文件系统
- 使用`mount`工具将分区挂载至目录使用
- 使用`blkid`查看分区UUID等信息
- 将配置写入/etc/fstab实现开机自动挂载

---

## Pratice

```bash
#fdisk使用交互式方案为块设备分区
fdisk /dev/sdb
#使用mkfs.ext4将分区格式化为ext4文件系统
mkfs.ext4 /dev/sdb1
#使用mount将分区挂载至目录
mount /dev/sdb1 /mnt/
```
