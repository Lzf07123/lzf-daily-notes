# 为内网镜像站添加域名解析

## Context

在使用过程中发现内网镜像站地址还属于DHCP动态分配，导致服务器重启后ip地址漂移，为解决这一问题，选择将服务器ip分配使用静态分配并添加域名解析方便后续更改ip地址

---

## Key Takeaway

- ubuntu24.04使用netplan配置文件管理网络配置
- 内网域名解析选择使用Adguardhome进行DNS重写

---

## Pratice

```bash
#netpan配置文件示例
network:
  version: 2
  ethernets:
    ens18:
      dhcp4: false
      addresses:
        - 192.168.1.88/24
      routes:
        - to: default
          via: 192.168.1.1
      nameservers:
        addresses: [192.168.1.1, 192.168.1.9]
```
