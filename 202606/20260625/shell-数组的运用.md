# shell-数组的运用

## Context

练习并理解shell脚本`数组`的基础用法。

## Key Takeaway

- 理解了数组的定义、访问方式
- 理解了数组值的的修改、添加、删除
- 理解了如何查询数组的长度、获取数组下标索引

> 数组使用下标索引的方式标记每个字符/字符串的位置，可以通过`${数组[下标]}`的方式访问数组内的字符/字符串

## Pratice

```bash
#!/bin/bash

arr=("系统版本" "主机名" "登陆用户" "外部IP地址" "运行时间" "总内存")

echo "数组：${arr[@]}"
echo "获取数组字符/字符串个数：${#arr[@]}"
echo "获取数组角标：${!arr[@]}"
echo "数组内第一个元素：${arr[0]}"
echo "数组内第二个元素：${arr[1]}"
echo '"删除数组内第一个元素：unset ${arr[0]}"'
unset arr[0]
echo "数组：${arr[@]}"
echo "添加值：arr+='Test'"
arr+="Test"
echo "数组：${arr[@]}"

```
