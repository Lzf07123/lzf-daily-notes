## Pratice

```bash
- 匹配系统版本
    grep -i "PRETTY_NAME" /etc/os-release | grep -oE '[^"=PRETTY_NAME]*'
- 匹配中国大陆11位手机号
    b1[3-9]\d{9}\b
- 匹配电子邮箱
    \w+@\w+(\.\w+)+\b
- 匹配年月日
    \d{4}-\d{2}-\d{2}
```
