# 第一次使用claude辅助完成旅游计划的编写

## Context

想在暑假前与舍友一起去厦门游玩一下，我负责旅行规划的编写，一想到如果全部要手写的话就会非常枯燥，突然想起人工智能在搜寻资料这方面非常契合，故使用claude code辅助编写旅行规划。

> claude适合处理繁琐的任务，写代码、写计划都非常合适

---

## Key Takeaway

经过六轮迭代后产出两个计划文档，在计划完成后对本次规划编写颇有感受，旅游规划开始前的提示词编写不够全面，没有正确认识到自己真实的需求导致‘做一步看一步’，耽误了很多时间，下次使用ai辅助工作时需提前明确自己的需求。

---

## Pratice

```markdown
# 厦门 3 天 2 晚旅行规划书

> 4 位大学生 2026 年 7 月初厦门旅行，两份规划：Plan A（民宿路线）与 Plan B（公寓路线）。

## 输出文件

| 文件                 | 定位                            | 路线逻辑                |
| -------------------- | ------------------------------- | ----------------------- |
| `Plan-A-民宿路线.md` | 以曾厝垵/环岛路民宿为中心       | 岛内自东向西一条龙      |
| `Plan-B-公寓路线.md` | 以集美/彩虹沙滩 LOFT 公寓为中心 | 岛外→岛内自北向南一条龙 |

---

## 规划内容

| 模块       | 说明                                                  |
| ---------- | ----------------------------------------------------- |
| 行程总览   | 3 天路线表格 + ASCII 地理逻辑图                       |
| 住宿推荐   | 首推/次推/备选，均附 Trip.com、携程、艺龙真实预订链接 |
| 每日路线   | 自由时间制，仅列路线流+实用信息+人均费用              |
| 费用预算   | 分项汇总，含学生证省钱提示                            |
| 避坑指南   | 10 条实用提醒（船票、预约、防晒、购物等）             |
| 美食地图   | 按位置分类的必吃清单+参考价格                         |
| 景点导航   | 全部景点的高德地图 `uri.amap.com/marker` 跳转链接     |
| APP/公众号 | 必备工具清单                                          |

---

## 关键设计原则

- **时间自由**：不设硬性时间表，仅保留景区运营时段等客观信息
- **中途可换宿**：Day 2 返回后可选换住更方便次日出发的位置
- **不走回头路**：每日景点沿地理方向线性串联
- **真实链接**：所有景点配高德地图坐标跳转；所有住宿配平台预订页

## 修改记录

### 第一轮：初始创建

- 搜罗厦门热门景点、路线、住宿，输出 4 天 3 晚 Markdown 规划
- 按 README 提供的 14 个携程意向房源分类为民宿（9 个）和公寓（5 个）
- Plan A 以曾厝垵民宿为中心，Plan B 以集美公寓为中心
- 使用 Markdown 格式化技能美化排版

### 第二轮：精简说明

- 去掉「设计理念」元描述
- 去掉末尾总结性段落
- 保留纯旅行实用信息

### 第三轮：真实链接替换

- 住宿推荐全部替换为搜索验证过的真实房源（Trip.com/携程/艺龙）
- 每个住宿均附可点击预订链接

### 第四轮：时间自由

- 去掉所有行程钟点（08:00、14:00 等）
- 「每日详细行程」→「每日行程路线」
- 代码块时间线 → 简洁路线概述（`> 路线：...`）

### 第五轮：3 天 2 晚

- 4 天 3 晚 → 3 天 2 晚，原 Day 3+4 合并
- 增加中途换宿策略（Day 2 可选换住）
- 费用预算同步缩减

### 第六轮：去除人数配置

- 「三男一女，含一对情侣」→「大学生结伴出游」
- 删除「情侣一间+朋友一间」「情侣上层+朋友下层」等分配建议
- 删除「÷ 4 人」「适合 4 人入住」等人数引用

### 第七轮：全景地图链接

- 路线中所有景点首次出现附高德 `uri.amap.com/marker` 链接
- 交通节点（厦门站/北站/地铁站）附链接
- 鼓浪屿子景点（八卦楼/龙头路/港仔后沙滩）附链接
- 集美学村子景点（龙舟池/鳌园/陈嘉庚纪念馆）附链接
- 美食地图位置（思北/局口街）附链接
- 末尾「景点导航速查」表集中索引（Plan A 24 个，Plan B 27 个）

### 最终状态

| 住宿类型     | 曾厝垵/环岛路民宿 | 集美/彩虹沙滩 LOFT 公寓 |
```
